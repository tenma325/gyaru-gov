# ギャル×行政 / GYARU-GOV

> 行政 × ギャルのコンセプトポータル。『その着せ替え人形は恋をする』の喜多川海夢をモチーフにしたドット絵マスコット「ガル子」が案内する、架空の自治体サイト。

## Live Demo

https://tenma325.github.io/gyaru-gov/

## コンセプト

「行政だから難しい」という先入観をやわらげるため、固い行政と開放的なギャル文化のギャップをデザインとして活性化。
パステルトーン × グランジテクスチャの融合で、老若男女が気軽に立ち寄れる雰囲気を目指している。

## マスコット

- **名前**: ガル子
- **役職**: ギャル行政課 主任
- **ビジュアル元**: TVアニメ『その着せ替え人形は恋をする』喜多川海夢（公式サイト: https://bisquedoll-anime.com/1st/character/）をモチーフにしたドット絵デフォルメ

## Stack

- Vite 8
- React 19
- TypeScript
- Framer Motion

## Design System

パステルカラー（コーラル、ラベンダー、ミント、レモン、スカイ）をアクセントに、グランジ／コラージュの質感を加えたデザインシステム。
詳細はプロジェクトルートの `Modelfile` および `src/index.css` を参照。

## ローカル開発

```bash
npm install
npm run dev
```

ビルド:

```bash
npm run build
npm run preview
```
